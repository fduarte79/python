# Entrada de dados via teclada
```
>>> x = input('Digite o valor: ')
Digite o valor: 5
>>> x
'5'
```

# Saída de dados
```
>>> x
5
>>> print(x)
5
```

# Saída formatada
```
>>> linguagem = 'Python'
>>> print(f'{linguagem} é fácil')
Python é fácil
```

# Saída posicional
```
>>> linguagem = 'Python'
>>> dificuldade = 'fácil'
>>>
>>> print("{} é {}".format(linguagem, dificuldade))
Python é fácil
```
