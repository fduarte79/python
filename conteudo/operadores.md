# Operadores 

## Aritméticos
```
>>> 22 + 7    # adição
29

>>> 22 - 7    # subtração
15

>>> 22 * 7    # multiplicação
154

>>> 22 ** 7   # potênciação
2494357888

 22 / 7       # divisão
3.142857142857143

>>> 22 % 7    # módulo (resto)
1

>>> 22 // 7   # quociente (parte inteira)
3
```
## Relacionais
```
>>> 3 < 4    # menor
True
>>> 4 <= 4   # menor ou igual
True
>>> 2 >  4   # maior
False
>>> 2 >= 4   # maior ou igual
False
>>> 0 == 1   # igual
False
>>> 0 != 1   # diferente
True
```
## Atribuição
```
>>> variavel = 0    # atribuição simples

>>> variavel = 1    # atribuição com operação     
>>> variavel += 2   
>>> variavel
3
```
