# Operadores 

## Aritméticos
```
>>> 22 + 7    # adição
29
>>> 22 - 7    # subtração
15
>>> 22 * 7    # multiplicação
154
>>> 22 ** 7   # potênciação
2494357888
 22 / 7       # divisão
3.142857142857143
>>> 22 % 7    # módulo (resto)
1
>>> 22 // 7   # quociente (parte inteira)
3
```
## Relacionais
```
>>> 3 < 4    # menor
True
>>> 4 <= 4   # menor ou igual
True
>>> 2 >  4   # maior
False
>>> 2 >= 4   # maior ou igual
False
>>> 0 == 1   # igual
False
>>> 0 != 1   # diferente
True
```
## Atribuição
```
>>> variavel = 1    # atribuição simples
>>> variavel += 2   # atribuição com operação  
>>> variavel
3
```

## Lógicos
```
>>> True and True   # E
True
>>> True or False   # OU
True
>>> not True        # NEGADO
False
```

## *Bitwise* (operador de bit)
```
>>> True & True    # E lógico
True
>>> True | False   # OU lógico
True
>>> bin(5)         # XOR (eXclusive OR)
'0b101'            # compara bit a bit,
>>> bin(2)         # e ambos devem ser
'0b10'             # diferentes
>>> 5^2
7
>>> bin(7)
'0b111'
>>> bin(5)        # ~ NOT 
'0b101'           # (inverte todos os bits)
>>> ~5
-6
>>> bin(-6)
'-0b110'

```
